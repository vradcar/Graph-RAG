**`run_query(question: str, depth: int = 2) -> str` function:**
1. Load settings via `load_settings()`
2. Get `neo4j_uri`, `neo4j_user`, `neo4j_password` from `settings["graph"]`, with `os.getenv("NEO4J_PASSWORD", settings["graph"]["neo4j_password"])` override
3. Get `model` from `settings["llm"]["model"]` and `provider` from `settings["llm"].get("provider", "groq")`
4. Build instructor client with error handling (try/except ValueError, print to stderr, sys.exit(1))
5. Open `Neo4jGraphStore` as context manager
6. Call `load_all_node_ids(store)` -- print count to stderr
7. Call `graph_retrieve(store, client, model, question, known_ids, depth=depth)` to get triples
8. Call `generate_answer(client, model, question, triples)` to get QueryAnswer
9. Return `format_answer(answer)`

Also expose a helper `run_query_structured(question, depth)` that returns the raw `QueryAnswer` object (for `evaluate.py` to use).

**`main()` function:**
- argparse with `--question` (required), `--depth` (int, default=2)
- Remove `--mode` and `--top-k` flags entirely (graph-only pipeline now)
- Call `run_query` and print result

**`if __name__ == "__main__": main()`**

No hardcoded model names anywhere. Model comes from `settings["llm"]["model"]` only (QUERY-06).

**`run_eval(queries_path: str, output_path: str) -> None`:**
1. Load settings, build Neo4jGraphStore (context manager), build instructor client
2. Load `known_ids = load_all_node_ids(store)`
3. Read queries JSON from `queries_path`
4. For each query:
   - `start = time.perf_counter()`
   - Call `graph_retrieve(store, client, model, question, known_ids, depth=depth)` for triples
   - Call `generate_answer(client, model, question, triples)` for QueryAnswer
   - `latency = time.perf_counter() - start`
   - Append result dict: `question`, `depth`, `answer` (prose), `evidence` (list of dicts via `.model_dump()`), `not_found`, `latency_sec`, `expected`
5. Write results JSON to `output_path`
6. Print summary: `Evaluated {N} queries. Saved to {output_path}`
